# IT Professional Alignment Chart

Interactive alignment chart that maps IT professional archetypes across four dimensions:

- **Judgment** (good ↔ bad)
- **AI Usage** (uses AI ↔ doesn't use AI)
- **IQ** (high ↔ low)
- **Work ethic** (hyperactive ↔ lazy)

Based on the Hockenmaier × Recuenco framework.

## Archetypes

| Archetype | Profile |
|-----------|---------|
| **Linus Torvalds** | High IQ · No necesita IA · Criterio sólido |
| **Twitter Autist** | Hiperactivo · Power user de IA · Buen criterio |
| **Turbo Brains** | Usa IA con criterio · Productividad alta |
| **Steady Hands** | Criterio sólido · Métodos tradicionales |
| **NPC** | Mediano en todo · No destaca |
| **Slop Cannon** | Genera basura con IA · Sin filtro |
| **Dead Weight** | Mal juicio · Aporta poco |
| **Prompt Goblin** | Cree que promptear es trabajar |
| **Office Furniture** | Parte del mobiliario · Calentando silla |
